Atlar Bank API

Welcome to the Atlar Bank API. This API is used to manage customers, accounts, and transactions.

To get started, please refer to the documentation.

    A list of example endpoints:

  • GET /api/v1/accounts
    Example response [ { "accountNumber": "54400001111", "currency": "SEK", "ownerName": "TEST Customer", "balance": { "amount": "3865371.31", "currency": "SEK", "date": "2018-12-17" } } ]
  • GET /api/v1/accounts/54400001111
    Example response [ { "accountNumber": "54400001111", "currency": "SEK", "ownerName": "TEST Customer", "balance": { "amount": "3865371.31", "currency": "SEK", "date": "2018-12-17" } } ]
  • GET /api/v1/accounts/54400001111/transactions
    Example response [ { "accountNumber": "54400001111", "entryReference": "LBE5419-0186-0029234", "amountDetails": { "amount": "242041.00", "currency": "SEK" }, "currency": "SEK", "bookingDate": "2018-12-17", "credit": false, "debit": true, "remittanceInformation": { "referenceInfo": { "msgID": "BF3970707", "accountServicerReference": "STOESI05181210113433580558000001", "paymentInformationID": "BB5806984", "instructionID": "TR40552142", "endToEndID": "EE32683251" } } }, ]
  • PATCH /api/v1/accounts/54400001111
    Example request { "ownerName": "Daisy", "balanceAmount": "2000" }
    Example response [ { "accountNumber": "54400001111", "currency": "SEK", "ownerName": "Daisy", "balance": { "amount": "2000", "currency": "SEK", "date": "2018-12-17" } } ]
  • DELETE /api/v1/accounts/54400001111
    Example request { "accountNumber": "54400001111" }
    Example response { "message": "Account with account number 54400001111 has been deleted", "accountNumber": "54400001111", "deleted": true }
